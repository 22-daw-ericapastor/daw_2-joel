Alertas

Las alertas se crean con la clase .alert, seguidas de las clases de colores contextuales:

Satisfactorio! Este cuadro de alerta podría indicar una acción exitosa o positiva.
Info! Este cuadro de alerta podría indicar un cambio o acción informativa neutral.
Advertencia! Este cuadro de alerta podría indicar una advertencia que podría necesitar atención.
Peligro!Este cuadro de alerta podría indicar una acción peligrosa o potencialmente negativa.
Primario! Indica una acción importante.
Secundario! Indica una acción ligeramente menos importante.
Oscuro/Dark! Alerta gris oscuro/Dark grey alert.
Ligero! Aleta gris lumninoso/Light grey alert.
