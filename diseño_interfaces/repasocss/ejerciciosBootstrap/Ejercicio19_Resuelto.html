Ejemplo Navbar Básico

Una barra de navegación es un encabezado de navegación que se coloca en la parte superior de la página.

La clase navbar-expand-xl | lg | md | sm determina cuándo la barra de navegación debe apilarse verticalmente (en pantallas extragrandes, grandes, medianas o pequeñas).

Navbar de Colres

Use cualquiera de las clases de color .bg para agregar un color de fondo a la barra de navegación.

Consejo: Agregue un color de texto blanco a todos los enlaces en la barra de navegación con la clase .navbar-dark, o use la clase .navbar-light para agregar un color de texto negro.

Navbar Colapsable

En este ejemplo, la barra de navegación se oculta en pantallas pequeñas y se reemplaza por un botón en la esquina superior derecha (intente cambiar el tamaño de esta ventana).

Only when the button is clicked, the navigation bar will be displayed.

Sugerencia: También puede eliminar la clase .navbar-expand-md para ocultar SIEMPRE los enlaces de la barra de navegación y mostrar el botón de palanca.
