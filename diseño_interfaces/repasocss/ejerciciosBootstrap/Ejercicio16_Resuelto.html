Desplegables

La clase .dropdown se usa para indicar un menú desplegable.

Usa la clase .downdown-menu para construir realmente el menú desplegable.

Para abrir el menú desplegable, use un botón o un enlace con una clase de .downdown-toggle y data-toggle = "dropdown".
