Barra de Progreso Básica

Para crear una barra de progreso predeterminada, agregaremos la clase .progress a un elemento contenedor y agregue la clase .progress-bar a su elemento secundario. Usaremos la propiedad CSS width para establecer el ancho de la barra de progreso.
